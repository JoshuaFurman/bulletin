# Infinite Canvas Bulletin Board

An infinite canvas application built with Next.js, React Flow, Tailwind CSS, and shadcn/ui.

## Features

- **Infinite Canvas**: Pan and zoom freely.
- **Cards**: Create, edit, update, delete, and move cards.
- **Modern UI**: Styled with Tailwind CSS and shadcn/ui components.
- **Interactive**: Double-click cards to edit content.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- [Next.js](https://nextjs.org/)
- [React Flow (@xyflow/react)](https://reactflow.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)
- [Lucide React](https://lucide.dev/) (Icons)
